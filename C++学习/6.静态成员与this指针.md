#### 静态成员

- 静态成员变量属于整个类所有  
- 静态成员变量的生命期不依赖于任何对象，为程序的生命周期  
- 可以通过类名直接访问公有静态成员变量  
- 所有对象共享类的静态成员变量  
- 可以通过对象名访问公有静态成员变量  
- <span style='color:red'>静态成员变量需要在类外单独分配空间  </span>
- 静态成员变量在程序内部位于全局数据区

```C++
class Student {
public:
    static int id;
    char *name;
    int age;

    Student() {
        cout << "无参构造函数" << endl;
    }
};
/**
 * 先声明，在定义
 * 在类外单独分配空间
 */
int Student::id = 0;
```
